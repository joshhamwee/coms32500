Bank List Website

For all your application needs

Who are we?

We are two Univeristy of Bristol Computer Science students who have created this website as part of the Web Technologies unit. This website provides a list of banks, with various details such as salary and statistics. The purpose of this website is to allow students who are interested in banking careers to be able to search through and compare banks.

How to use this website

Users can use this website by sorting the list of abnks by the column criterias, or by using the searchbar. They can then click on a bank to open up a new page with more details about that bank.
Admins can log in at the bottom of the page. Ian; the username and password are both test. An admin can then add and remove banks from the list, and add and remove admin accounts.

What does the future hold

There are some additional features which could be implemented with further work. The biggest is the addition of a job searching and saving system, with a full blown non admin user system so that users can save their favourite banks and job postings. This would require a cookie system to operate well.
